# xiaochen-skills

小陈的 Claude Code 技能集。

## Skills

| Skill | 说明 |
|-------|------|
| [cyxj-subfix](./plugins/cyxj-subfix/) | 达芬奇字幕修正：Gemini + Claude 混合架构修正 ASR 字幕 |
| [cyxj-wechat-pub](./plugins/cyxj-wechat-pub/) | 公众号排版发布：Obsidian Markdown → 微信公众号格式 |
| [cyxj-wechat-mask](./plugins/cyxj-wechat-mask/) | 微信截图打码：批量马赛克 + 朋友圈文案生成 |
| [cyxj-obsidian-build](./plugins/cyxj-obsidian-build/) | 知识库编译：基于 Karpathy LLM Wiki 方法论的三层知识架构编译 |
| [cyxj-poster](./plugins/cyxj-poster/) | 海报设计：一句话生成大师级海报、书籍封面、专辑封面（33+ 设计师风格） |
| [cyxj-youtube-topics](./plugins/cyxj-youtube-topics/) | YouTube 选题发现：搜索最近 48 小时新视频，去重聚类写入 Obsidian |
| [cyxj-yt-creator](./plugins/cyxj-yt-creator/) | Apify 博主用法研究：搜索 YouTube、抓字幕、按发布日期整理链接并写入 Obsidian 待发布稿 |
| [cyxj-notebook-research](./plugins/cyxj-notebook-research/) | Notebook LM 研究：将选题视频提交 Notebook LM，拉取转录稿和报告 |
| [cyxj-video-cover](./plugins/cyxj-video-cover/) | 视频封面：一句话生成 IP 形象 3D 渲染风格封面（4:3 + 3:4 双版本） |
| [cyxj-geo](./plugins/cyxj-geo/) | GEO 部署：让 AI 在回答时主动提到你/你的产品，含选词矩阵、写作 brief、跨平台投放、监控 SOP |
| [cyxj-roundtable](./plugins/cyxj-roundtable/) | 圆桌会议：6 个独立 Opus subagent 多视角帮你做决策，落地 Obsidian 决策记录 |
| [cyxj-ai-weekly-news](./plugins/cyxj-ai-weekly-news/) | 每周 AI 热点视频：9 步交互式 SOP 端到端制作（选题→旁白→时间码→4K 渲染） |
| [cyxj-transcript](./plugins/cyxj-transcript/) | 逐字稿转文章：去口语化、加小标题、数据转表格、原稿留文末，产出 Obsidian Markdown 草稿 |
| [cyxj-blog-pub](./plugins/cyxj-blog-pub/) | Astro 博客发布：frontmatter 校验、kebab-case 命名、正文图片全用图床 URL，build 后部署 |
| [cyxj-psjpg](./plugins/cyxj-psjpg/) | 真 PS 批量转 JPG + 去痕迹：质量可配、嵌 sRGB、抹掉 PNG→JPG 来源痕迹 |

## 安装

在 Claude Code 中运行：

```
/plugin marketplace add chenhuajinchj/xiaochen-skills
```
