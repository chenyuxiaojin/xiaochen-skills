# 6 平台标题+简介规格(陈与小金)

主平台:YouTube / B站 / 抖音。分发平台:视频号 / TikTok / 小红书。
标题+简介**成对**交付。写完逐条自检:体裁匹配?数字有源?没有「不是…而是…」?

## YouTube(主)

- **标题**:SEO 向,搜索关键词前置(中文词 + 英文术语双写,如「循环工程 Loop Engineering」),
  可用 `|` 接副题;疑问句/获得感句式好用。
- **简介**:可长。结构 = 2-3 段内容概述(含核心数字与诚实短板)→ 空行 → `章节:` 时间戳列表
  → 空行 → `#标签`。
- **章节时间戳**:第一条必须 `00:00`;每条从**最终导出字幕**核对,格式 `MM:SS 章节名`;
  8-10 条为宜,章节名用观众语言不用内部黑话。

## B站(主)

- **标题**:可长、口语化,「讲明白/带你搭/我用 XX 给你讲清楚」这类获得感句式;爆火词可以带。
- **简介**:比 YouTube 短,2-3 句讲清这期干什么 + 一句诚实短板;结尾固定带三连引导
  (「觉得有用的话,三连是对我最大的支持」);分段进度可复用 YouTube 章节戳(放置顶评论)。

## 抖音(主)

- **标题/文案**:第一行就是钩子,短、口语、反差(「你睡觉,AI 干活」「你还在一句句问 AI?」);
  第二行 1-2 句展开;**末尾堆标签**(4-6 个,主题词+工具词+泛词混搭)。
- 版面小,诚实短板不必塞;禁绝对化用语(保证/一定/唯一)——有限流前科。

## 视频号(分发)

- 微信生态,**干净、少标签**:1-2 句讲清内容+核心数字即可,最多带 1 个标签或不带。

## 小红书(分发)

- **标题 ≤20 字**(硬限制,交付前数一遍)。
- **正文**:emoji 分点种草风(✅/🔁/⚠️/📌),3-5 个点;必带一条诚实点(可信度);
  收藏引导(「记得收藏慢慢看」);末尾 4-6 个标签。

## TikTok(分发)

- 默认英文(国际向);**视频是中文音轨时先问用户投哪个受众**:华语区 → 直接复用抖音版;
  国际区 → 英文标题+简介,英文里保留精确数字,标签用 #LoopEngineering #ClaudeCode 这类英文 tag。

## 常用标签池

`#ClaudeCode` `#AI自动化` `#效率工具` `#AI` + 当期主题词(中英各备一个)。
英文:`#AIAutomation` `#AIWorkflow` + 当期英文术语。
