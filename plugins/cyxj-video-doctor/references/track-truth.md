# 赛道真相（cyxj-hook / cyxj-content 共享单源）

> 两个 skill 的共同赛道前提。**改这里，两边同时生效**——不要再往各自 SKILL.md 里复制数字。
> 具体数字的权威来源是 `/cyxj-data-review` 的最新复盘；本文件只留定性结论。

## 赛道定义

- 这不是 15-60 秒短视频带货，是 **3 分钟+ 的知识口播 / 演示视频**。
- 目标是 **抖音精选**。
- 主播定位：**把复杂讲到小白能懂**的知识博主（非程序员用 Claude Code / Codex 做一切）；受众是「想用好某个工具/方法的普通人」。
- 抖音后台官方标记（平台给的，非我们推断）：垂类 = **科技/职场**（账号周期汇总导出）；单条视频是否进精选看**作品详情页**的「已被标记为抖音精选内容」卡片——这是唯一可见通道，xlsx 导出与数据中心页面均无此字段。

## 精选实证（2026-07-11 · 28 条公开逐条实查，不再是推断）

- 账号已有 **7 条进精选**，全部为 2026-06-05 后发布；此前作品全未标记（含高收藏率老教程）→ 精选评估约 6 月起才覆盖本账号，老视频不回溯，别拿老视频没进自我怀疑。
- **精选收「可学的实操/教程/演示」，拒「热点/资讯」**：6 月后仅有的两条落选都是热点型（含账号播放最高的爆款）；反之 2 千多播放的小教程也能进。不卡时长、不卡播放量、不卡收藏率下限。
- **精选的直接流量很小**（单条扶持 ≤ 千余次播放，现金激励元级）——它的价值是「官方认可内容特征」的信号与可能的加权，**不是流量主引擎**。别把「进精选」当流量策略卖点，把它当质检合格章。

## 数据校准（快照 2026-07-11，只留定性结论）

- **北极星 = 收藏率（收藏/播放），不是播放量。** 每条目标 3%+。它驱动长尾与涨粉；精选评估看内容类型、不卡收藏率，但两者指向同一件事：可学性。
- 长视频完播率% 天然个位数，**不判生死**；看绝对观看时长。
- 知识长视频（5min+ 可操作/清单/实测类）的收藏率显著高于热点内容；**5min+ 可照做类是当前最优解**，也是涨粉主引擎（每播放涨粉效率比热点爆款高一个数量级）。
- **同题材也分形态：「带你做完」≠「讲给你听」。** 2026-06 底~07 初连续三条讲解/观点形态的新片，收藏率跌到自家「可照做」教程基准的一半以下——选题对不救形态错。硬规格：步骤清单 + 可复制资产（prompt/命令/仓库链接）+ 口播收藏钩子。
- **热点爆款仍是涨粉陷阱，但「爆完即停」不再绝对**（2026-07 复盘修正）：带一定收藏价值的热点也可能拖出月级长尾、贡献大播放；然而换粉效率仍差一个数量级、且被精选拒收——可当触达副线，不当主线。
- **不要建议「拆成短视频」**：多平台分发可以，但母体不动。
- 观众收藏的是「以后能照着做的东西」（清单 / 带数字的实测 / 可抄模板），不收藏「这瓜真大」的吐槽。
