# cyxj-ps-export

让本机 Photoshop 批量把一个文件夹的图统一过一遍。

## 用法

```bash
${CLAUDE_PLUGIN_ROOT}/scripts/batch_export.sh <源目录> [输出目录] [质量 1-12]
```

| 参数 | 默认 |
|------|------|
| 源目录 | 必填 |
| 输出目录 | `<源目录>_ps` |
| 质量 | `12`（PS 最高档）|

## 工作流

1. 找用户给的源目录（明确路径直接用；模糊指代去对话上下文里找；找不到就问）
2. 跑脚本
3. 转述结果：成功/失败、输出位置、同名覆盖提示

## 已知限制

- 仅 macOS（用 osascript 驱动 PS）
- 需要本机已装 Adobe Photoshop
- 跑的过程中 PS 被占用，你不能同时手动用 PS
- 99 张 1024×1536 实测约 35 秒
- 源目录里如果有 `foo.png` 和 `foo.jpeg` 同基础名，输出会互相覆盖（脚本会提示）
- 原图所有 metadata 会被 PS 重写（这是 PS 重编码的自然结果）
