# 自动化监控参考(阶段四)

> 本文件是 cyxj-geo 阶段四(数据监控)的参考资料。手动版 SOP 在 SKILL.md 里;目标词 >5 个、考虑上自动化时读取本文件。

## 自动化版监控 SOP(目标词 >5 个再考虑,先看坑)

- 工具:Playwright + `playwright-stealth` + `storage_state`(保存登录态)
- 每天定时跑目标搜索词,输出 CSV:词 / 时间 / 是否引用 / 来源平台 / 占比
- **实操坑(先看再决定要不要做)**:
  1. 豆包登录墙较严(手机号/扫码),需先手动登录一次保存 cookie
  2. 三家都做反爬:JS 指纹、行为检测、签名(豆包有 `a_bogus`)
  3. 大规模高频跑会触发限流甚至封号
  4. 建议先**手动跑两周**摸清频率上限,再决定要不要自动化
  5. **不要用 doubao-2api 类 GitHub 逆向 API 项目**(违反平台条款,与本方案的白帽合规立场冲突,明确不推荐)

## 监控数据示例

```
目标词:北京 AI 产品经理培训机构推荐有哪些
日期:2026-04-26
豆包结果:
  - 总引用数 10 条
  - 我方文章被引用 4 条(占比 40%)
  - 来源:今日头条 2 篇 + 抖音 1 篇 + 自建文章 1 篇
  - 排名:第 3 位(在"且曼"之后)
```
