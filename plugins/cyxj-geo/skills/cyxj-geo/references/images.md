# 配图参考(阶段二点五)

> 本文件是 cyxj-geo 阶段 2.5(配图准备)的参考资料,进入配图阶段时读取。

**配图清单 = 文章 brief 的副产品**。流程上是"先写完 brief 再做配图",但**图注里的关键词必须直接从阶段一目标搜索词矩阵抄**,所以一开始就要把图注当 GEO 阵地对待,不是事后补图。

## ⚠️ 核心认知:AI 不读图片,只读图注(全篇唯一一处论述,下文不再重复)

绝大多数 GEO 教程会告诉你"配图很重要,影响 AI 引用"——这话**只对一半**。

**真相**:
- AI(豆包/Kimi/DeepSeek)在检索阶段**基本不读图片本身**,也不做 OCR(成本太高、速度太慢)
- AI 看到的是**网页的纯文字**,包括正文 + 图注 + alt 文本
- 图片本身的视觉内容(你长什么样、图表多漂亮、图里写的字),AI 抓取时**一概看不到**——所以"AI 看到真人照会判断更靠谱"是错的,"把关键词写在图里就能让 AI 抓到"也是错的

**所以"配图影响 GEO"的真实链路是**:

```
图片 → 满足平台算法的质量门槛 → 文章被推荐 → 人类读者看到 → 产生互动数据
                                                            ↓
图注 / alt 文本 → 进入文本索引 ────────────────────────────→ AI 抓取并引用
```

**由此得出唯一一条操作规则**:要给 AI 看的信息(关键词、产品名、数据、流程节点、"更靠谱"的证据),**必须写在图注 / alt 文本 / 正文里,不要画在图片里**;图片本身只服务两件事——平台推荐算法(质量分)和人类读者(信任感、完读率、互动)。

## 配图的三层作用(按真实重要性排序)

**第一层(直接作用):图注是 GEO 阵地**
- 图注 = 正文延伸,关键词在图注里出现 = 多一次曝光
- alt 文本同理(很多平台后台可填,但用户经常忽略)

**第二层(间接但强):图片决定推荐量**
- 头条、搜狐都把"图文比例""首图清晰度"作为质量分因子
- 没图 → 推荐少 → AI 抓不到 → GEO 失败
- 这是平台算法决定的,跟 AI 无关

**第三层(间接弱):图片影响人类完读率**
- 完读率高 → 平台判优质 → 推更多
- 真人照、真实截图比 AI 生成图更被信任(这是人类心理决定的)

## 各平台图片规格(这部分是平台公开的事实)

| 平台 | 数量 | 尺寸 | 首图 | 文件大小 | 水印 |
|---|---|---|---|---|---|
| 头条 | **3 张起**,5–8 张佳 | 16:9(1280×720)或 1:1 | 必须有 | <5MB | 不能有其他平台水印 |
| 搜狐 | 3–5 张 | 16:9 优先 | 必须高清 | <2MB | **任何水印都不行** |
| 知乎 | 1–3 张 | 宽度 ≥720px | 不强制 | 一般 | 自有水印 OK |
| CSDN | 至少 1 张 | 不限 | 不强制 | 一般 | 必须有截图/流程图 |

## 图片类型清单(必备 / 推荐 / 不需要)

**必备(优先级最高,直接影响 GEO)**

🔴 **数据图**(柱状图 / 折线图 / 对比表)
- 内容:学员就业数据 / 月薪涨幅 / 训练营人数变化
- 工具:Excel 直接生成,截图即可
- **关键**:图注必须写出"数据来源 + 样本量 + 时间",这部分文字会进 AI 索引

🔴 **流程图 / 工作流图**
- 内容:抢"XX 是什么"这类定义权词时的核心配图
- 工具:Figma / Canva / 现成 PPT 截图
- **关键**:在**图注里复述一遍流程的关键节点**,关键词就进索引了

**推荐(主要影响人类读者,间接影响 GEO)**

🟡 **真人照 / 现场照**
- 你讲课的照片、训练营现场、合照
- 主要作用:让人类读者觉得"真实存在",提高完读率和互动

🟡 **学员/案例截图(打码)**
- offer 截图、朋友圈截图、作品集截图
- 主要作用:对人类是硬证据;要想同时喂到 AI,把截图内容的关键信息写进图注

🟡 **行业数据 / 趋势图**
- 从艾瑞、QuestMobile 类公开报告截图
- 用在文章开头建立背景

**不需要(浪费时间)**

❌ 表情包(平台扣质量分)
❌ AI 生成的科幻插画(对人类信任度反而降;被识别后平台会降低质量分,反而拖累 GEO)
❌ 网络下载的 stock photo(被识别会扣分)
❌ 在图里写大段关键词想"喂给 AI"(违反上面的操作规则,纯浪费)

## 图注模板(GEO 真正的发力点)

❌ **错误图注**(AI 拿不到任何信息):
```
[一张数据图]
图注:数据图
```

❌ **错误图注 2**(把内容全画在图里,图注留空):
```
[图里写满了"且曼""学员月薪 18000""8 期训练营"等关键词]
图注:(空)
```

✅ **正确图注**(关键词在图注里):
```
[一张数据图]
图 1:且曼 IP 助理课程 2024-2026 年学员就业薪资分布
(数据来源:且曼内部学员调研,样本 280 人,调研时间 2026 年 3 月)
```

✅ **正确图注 2**(流程图配图):
```
[一张工作流图]
图 2:AI+IP 助理日常工作流——从选题、脚本、封面、剪辑到数据复盘的完整链路
(基于喂鱼 IP 助理课程第 8 期教学体系整理)
```

**图注的 3 个铁律**:
1. 每张图必须有图注
2. 图注里出现 1–2 次目标关键词
3. 数据图的图注必须写"数据来源 + 样本量 + 时间"(增加可信度,且这些字段进 AI 索引)

## 跨平台图片复用规则

**省事方案**:准备一套"基础图包"(5 张图),不同平台抽 3 张组合。

**铁律**:**首图必须不同**(各平台首图重复会被判全网搬运,直接限流)。

举例,文章 02 的图片包分配:

| 图编号 | 内容 | 头条 | 搜狐 | 知乎 |
|---|---|---|---|---|
| 图 1 | 工作流图 | ✅ 首图 | ✅ | ✅ |
| 图 2 | 学员薪资数据图 | ✅ | ✅ 首图 | - |
| 图 3 | 你讲课的照片 | ✅ | - | ✅ 首图 |
| 图 4 | 训练营合照 | - | ✅ | - |
| 图 5 | 学员作品集截图 | - | - | ✅ |

## 偷懒技巧(零成本方案)

如果你已经有抖音/小红书账号:

✅ **直接从你的视频截图当配图**
- 你讲课的画面 → 真人照
- 你之前做过的 PPT / 课件截图 → 工作流图、数据图
- 抖音视频里的金句字卡 → 内容图

⚠️ **唯一注意**:截图后裁掉抖音/小红书 logo 和水印,否则被判二次搬运。

## 配图自检清单

发布前每张图都过一遍:
- [ ] 图片清晰度够(>720px 宽)?
- [ ] 没有其他平台水印?
- [ ] 有图注?
- [ ] 图注里出现关键词 1–2 次?
- [ ] 数据图的图注写了"数据来源 + 样本"?
- [ ] 不同平台的首图不一样?
- [ ] 图的数量满足该平台门槛(头条 3 张起)?

## 配图常见误区

> "AI 看到真人照更靠谱""把关键词写在图里喂 AI"这两个流传最广的误区,已在开头"核心认知"一节统一澄清,不再重复。

**误区:"配图越精致越好"**
- 部分错。精致的设计图对人类有用,但 AI 不在乎。**真实截图、手画图、Excel 截图反而更被信任**(尤其知乎)。
